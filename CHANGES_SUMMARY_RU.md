## 🔄 Как это работает теперь

```
1. Запуск сервера (на машине учителя):
   python -m server.gui.main_window
   ↓
   Сервер определяет свой IP (например, 192.168.1.1)
   ↓
   Слушает на 0.0.0.0:8000 (все интерфейсы)

2. Запуск агентов (на каждой рабочей станции):
   python -m agent.main
   ↓
   Слушают на 0.0.0.0:8001

3. Обнаружение агентов:
   Меню → ⚙️ Управление → 🔍 Поиск агентов
   ↓
   Сканирует диапазон подсети (192.168.1.0/24)
   ↓
   Находит все машины с портом 8001 открытым
   ↓
   Обновляет список рабочих станций

4. Управление:
   Можно отправлять команды любому агенту в сети
   по его IP адресу, независимо от того, где находится сервер
```

## 🚀 Как использовать

### Быстро:
1. Запустить `python test_network.py` - проверить конфигурацию
2. На сервере: `python -m server.gui.main_window`
3. На каждом ПК: `python -m agent.main`
4. В GUI: Меню → Поиск агентов

### Подробно:
- Читать `QUICK_START.md` для быстрого старта
- Читать `NETWORK_SETUP_RU.md` для полного руководства
