# 🎓 Classroom Manager - Система управления компьютерным классом

## ✨ Новое: Поддержка локальной сети!

## 🚀 Быстрый старт

### На машине учителя (Сервер):

```bash
# Клонировать проект
cd /Volumes/ADATA/PROGRAMMING/class-assistant-project/lab_manager

# Установить зависимости
pip install -r requirements.txt

# Запустить GUI сервера
python -m server.gui.main_window
```

### На каждой рабочей станции (Агент):

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить агент
python -m agent.main
```

## 💻 Системные требования

- **Python**: 3.8+
- **ОС**: Windows, macOS, Linux
- **Сеть**: Локальная сеть (Ethernet или WiFi)
- **Порты**: 8000 (сервер), 8001 (агенты)

## 🎮 Использование

### Запуск сервера

```bash
# Способ 1: Через GUI (Рекомендуется)
python -m server.gui.main_window

# Способ 2: Через API напрямую
python -m server.api

# Способ 3: На конкретном IP/порте
python -m server.api --host 0.0.0.0 --port 8000
```

### Запуск агента

```bash
# На каждой рабочей станции
python -m agent.main

# Или с указанием хоста/порта
python -m agent.main --host 0.0.0.0 --port 8001
```

### Автоматическое обнаружение агентов

1. Убедитесь, что все агенты запущены
2. В GUI перейдите: **⚙️ Управление → 🔍 Поиск агентов**
3. Приложение автоматически найдет все агенты в сети

### Управление рабочими станциями

После обнаружения агентов можно:
- 🔒 Блокировать экраны
- 🛑 Выключать компьютеры
- 🔄 Перезагружать системы
- 🌐 Управлять интернетом
- 👤 Применять профили работы

## 🧪 Диагностика

### Проверка соединения

```bash
# Проверить, что агент доступен
nc -zv <agent_ip> 8001

# Должно вывести: Connection to <agent_ip> port 8001 [tcp/*] succeeded!
```

### Запуск сетевых тестов

```bash
# Полный тест сетевой конфигурации
python test_network.py

# Этот скрипт проверит:
# ✅ Определение локального IP
# ✅ Диапазон сети
# ✅ Сканирование портов
# ✅ Обнаружение агентов
```

### Типичные проблемы

| Проблема | Решение |
|---------|--------|
| Агенты не найдены | Проверьте, запущены ли агенты (python -m agent.main) |
| Брандмауэр блокирует | Откройте порты 8000, 8001 в брандмауэре |
| Разные подсети | Убедитесь, что все машины в одной подсети (192.168.x.x) |
| Медленное подключение | Проверьте сетевое соединение, может быть WiFi |

## 📝 Примеры использования

### Python скрипт для обнаружения агентов

```python
from server.network_discovery import discover_agents_sync

# Найти все агенты на сети
agents = discover_agents_sync()
print(f"Найдено {len(agents)} агентов: {agents}")

# Вывод:
# Найдено 3 агентов: ['192.168.1.10', '192.168.1.11', '192.168.1.12']
```

### Отправка команды агенту

```python
import asyncio
from server.api_client import APIClient
from shared import schemas

async def send_command():
    client = APIClient("http://192.168.1.1:8000")
    
    # Заблокировать экран на рабочей станции
    command = schemas.CommandRequest(command="lock_screen")
    response = await client.send_command("192.168.1.10", command)
    print(response)

asyncio.run(send_command())
```
