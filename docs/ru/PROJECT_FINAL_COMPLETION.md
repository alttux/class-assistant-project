## 🚀 БЫСТРЫЙ СТАРТ

### Для Разработки
```bash
cd lab_manager
source ../.venv/bin/activate
python run_teacher_gui.py  # GUI учителя
python run_agent_gui.py    # GUI агента (в другом терминале)
python test_api.py         # Запуск тестов
```

### Для Распространения
```bash
# macOS (готово)
unzip ClassroomManager_macOS.zip
./ClassroomManager-Teacher

# Windows (собрать на Windows)
python build_windows.py

# Linux (собрать на Linux)
python build_linux.py
```
