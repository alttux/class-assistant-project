## 🎯 Что Это

Classroom Manager - система управления IT классом с графическими интерфейсами для учителя и учеников. Полностью автономные исполняемые файлы, созданные с помощью PyInstaller.

---

## 📥 Скачивание

### macOS
- **Файл**: `ClassroomManager_macOS.zip` (162 MB)
- **Содержимое**: Исполняемые файлы + инструкции
- **Требования**: macOS 10.15+

### Windows (готово к сборке)
- **Скрипт сборки**: `build_windows.py`
- **Результат**: `ClassroomManager_Windows.zip` (~150 MB)
- **Требования**: Windows 10/11

### Linux (готово к сборке)
- **Скрипт сборки**: `build_linux.py`
- **Результат**: `ClassroomManager_Linux.tar.gz` (~140 MB)
- **Требования**: Linux с GTK+

---

## 🚀 Установка и Запуск

### macOS
```bash
# 1. Распаковать
unzip ClassroomManager_macOS.zip

# 2. Сделать исполняемыми (если нужно)
chmod +x ClassroomManager-*

# 3. Запустить
./ClassroomManager-Teacher    # Для учителя
./ClassroomManager-Agent      # Для ученика
```

**Важно**: При первом запуске macOS заблокирует файлы. Правой кнопкой мыши → "Открыть" → "Открыть".

### Windows
```bash
# 1. Распаковать ZIP
# 2. Запустить
ClassroomManager_Teacher.exe    # Для учителя
ClassroomManager_Agent.exe      # Для ученика
```

### Linux
```bash
# 1. Распаковать
tar -xzf ClassroomManager_Linux.tar.gz

# 2. Сделать исполняемыми
chmod +x classroom-manager-*

# 3. Запустить
./classroom-manager-teacher    # Для учителя
./classroom-manager-agent      # Для ученика
```

## 🎮 Использование

### Для Учителя
1. Запустить `ClassroomManager-Teacher`
2. Нажать "▶️ Запустить сервер"
3. Использовать 4 вкладки:
   - **Мониторинг**: Просмотр ПК учеников
   - **Управление**: Отправка команд
   - **Профили**: Управление настройками
   - **Отчеты**: Просмотр логов и экспорт PDF

### Для Ученика
1. Запустить `ClassroomManager-Agent`
2. Приложение покажет:
   - Мониторинг CPU/RAM/Disk
   - Журнал событий
   - Статус подключения

## 🌐 Сетевая Конфигурация

### Порты
- **Сервер**: 8000 (HTTP)
- **Агент**: 8001 (HTTP)

### Файрвол
Разрешить входящие соединения на портах 8000 и 8001.

### Рабочие станции
- Учитель: Один ПК с запущенным сервером
- Ученики: Каждый ПК ученика запускает агента

## 🔧 Устранение Неисправностей

### macOS: "Не удается открыть"
```
Решение: Правой кнопкой → "Открыть" → Подтвердить
Или: xattr -rd com.apple.quarantine ClassroomManager-*
```

### Windows: Антивирус блокирует
```
Решение: Добавить исключение в антивирусе
Или: Отключить real-time защиту временно
```

### Linux: "Отсутствуют библиотеки"
```
Решение: Установить зависимости (см. выше)
Или: sudo apt-get install python3-pyqt6
```

### Сеть: Не подключается
```
Проверить:
- ПК в одной сети
- Порты 8000/8001 открыты
- Файрвол разрешает соединения
- IP адреса правильные
```
