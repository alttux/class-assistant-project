## 📡 API ЭНДПОИНТЫ (ПОДТВЕРЖДЕНО РАБОТАЮЩИМИ)

### Server API (http://localhost:8000)
- ✅ `GET /workstations/` - Получить все ПК
- ✅ `POST /workstations/` - Создать ПК
- ✅ `PUT /workstations/{id}/status` - Обновить статус
- ✅ `GET /profiles/` - Получить профили
- ✅ `POST /profiles/` - Создать профиль
- ✅ `GET /logs/` - Получить логи
- ✅ `POST /logs/` - Создать лог
- ✅ `GET /report` - Сгенерировать отчет
- ✅ `POST /export_report` - Экспорт в PDF
- ✅ `POST /command/{ip}` - Отправить команду
- ✅ `POST /apply_profile` - Применить профиль

### Agent API (http://localhost:8001)
- ✅ `GET /monitor` - Получить метрики
- ✅ `POST /command` - Выполнить команду
